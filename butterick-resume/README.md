# Butterick's Résumé

A résumé template based on the sample provided by Matthew Butterick in his web book [*Practical Typography: Résumés*](https://practicaltypography.com/resumes.html).

**NOTE:** The data within the template is extracted directly from the [original sample résumé](https://practicaltypography.com/samples/resume-after.pdf) and is used for demonstration purposes only.
